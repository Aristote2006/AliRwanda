# Favicon Placeholder

This directory should contain:
- favicon.png (32x32 or 48x48 PNG version of your logo icon)

## How to create:

1. Open logo-icon.svg in any browser
2. Take a screenshot or use a converter tool
3. Save as favicon.png in this directory

Or use online tools:
- https://favicon.io
- https://realfavicongenerator.net
